# Fake News Detection using NLP

## Project Overview
This project builds a machine learning model to classify news articles as real or fake using natural language processing techniques.
The goal of the project was to examine how different text preprocessing strategies affect model performance.
The dataset contains labeled news articles categorized as real or fake, enabling supervised machine learning models to learn distinguishing linguistic patterns.

## Techniques Used
- Text preprocessing (tokenization, stopword removal)
- Feature vectorization (CountVectorizer, TF-IDF)
- Multinomial Naive Bayes classification
- Model evaluation and comparison

## Tools
- Python
- Scikit-learn
- NLTK
- Pandas
- Google Colab

## Results
Multiple preprocessing configurations were tested to analyze how feature engineering decisions influence classification accuracy.
